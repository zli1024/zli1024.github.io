AlexNet

Paper: ImageNet classification with deep convolutional neural networks

Link: https://dl.acm.org/doi/10.1145/3065386

Milestone: 拉开卷积神经网络的序幕

Abstract

  1. 在ILSVRC-2010的120万张图片上训练深度卷积神经网络，获得最优结果，top-1和top-5 error分别为37.5%, 17%
  2. AlexNet由5个ConvNet和3个FC层组成
  3. 为加快训练，采用非饱和(non-saturating)激活函数–ReLU，两GPU并行训练
  4. 采用Dropout减轻过拟合
  5. 成果： 在ILSVRC-2012以超出第二名10.9个百分点成绩夺冠

Characteristics

ReLU的优点：

1 使网络训练更快（function简单）

2 防止梯度消失（vanishing）–> 对标Sigmoid\tanh等saturating nonlinearities函数会出现梯度消失的情况

3 使网络具有稀疏性

Local Response Normalization(LRN)

用处不大，后续论文实验证明了

Overlapping Pooling

包括Max Pooling 和 Average Pooling

VGG

Paper: Very Deep Convolutional Networks For Large-scale Image Recognition

Link: https://arxiv.org/abs/1409.1556

Results: Top-1 in the localisation Top-2 in the classification

Milestone: 开启小卷积核，深度卷积模型时代

Background:

  1. ILSVRC-2014

  2. Related work:

    AlexNet: ilsvrc-2012分类冠军，里程碑的CNN

    ZFNet: ILSVRC-2013分类冠军方法，对AlexNet改进

    OverFeat: ILSVRC-2013定位冠军，集分类、定位和检测于一体的卷积网络方法

    NiN architecture:

    • 提出用Global Average Pooling 代替FC层
    • 交替使用NiN块和步幅为2的最大池化层，逐步减小高宽和增大通道数
    • 最后使用全局平均池化层得到输出，其输入通道数是类别数

Abstract

  1. 研究深度对提升精度的影响
  2. 通过3*3卷积堆叠使用来增加网络深度
  3. 16-19层weight layers对效果显著
  4. 成果SOTA: Top-1 in the localisation Top-2 in the classification in ILSVRC
  5. 泛化性能好，对其他数据集均能达到SOTA

Architecture

Characteristics

  1. 堆叠使用3*3小卷积核

    –> 原因:：2个3x3 = 1个5x5 3个3x3=1个7x7

    –> 作用：堆叠使用增大感受野 / 增加非线性激活函数，增强特征抽象能力 / 减少训练参数

  2. 比别人”deep”

Training Tricks

Data Augmentation:

方法一：针对尺寸(Scale jittering)

  1. 按比例缩放图片至最小边S
  2. 随机位置裁剪出224*224区域
  3. 随机进行水平翻转

S的设置方法：

  1. 固定值：固定为256或384
  2. 随机值：每个batch的S在[256, 512]，实现尺度扰动

方法二：针对颜色

修改RGB通道的像素值，实现颜色扰动

Pretrained model initialization

深度神经网络对初始化敏感

  1. 当深度加深时，用浅层网络初始化

    如BCDE模型用A模型初始化

  2. Multi-scale训练时，用小尺度初始化

    S=384时，用S=256模型初始化

    S=[256, 512]时，用S=384模型初始化

也就是深层模型用浅层模型初始化，大尺度用小尺度初始化

Testing Tricks

Multi-scale evaluation:

图片等比例缩放至最短边为Q，设置三个Q，得到3种尺寸，然后输入三种尺寸得到输出取平均

方法一：当S为固定值时 Q=[S-32, S, S+32]

方法二：当S为随机值时 Q=[S_min, 0.5*(S_min + S_max), S_max]

Dense evaluation:

将FC层转换为卷积操作，变为全卷积网络

  1. 经过全卷积网络得到 NxNx100 特征图
  2. 在通道维度上spatially averaged(sum-pooled) 计算平均值，得到1*1000的输出向量

GoogLeNet-v1

Paper: Going deeper with convolutions

Link: https://arxiv.org/abs/1409.4842

Milestone: Google系列第一篇文章，首次提出多尺度卷积模块——inception module

Abstract

  1. 提出了inception module
  2. GoogLeNet(22 “weighted” layers)的结构设计不仅是提高精度，最重要是要利用network内的计算资源

Architecture

Inception Module:

这里用了1x1\3x3\5x5的多尺度卷积，因此能够提取到不同特征

GoogLeNet-v1 architecture:

将图表和结构图结合着一起来看 这里#3x3 reduce在inception module中实际上是1x1卷积，用于降维

Training Tricks

Auxiliary Classifier

辅助分类器：

在inception4a和inception4d增加两个辅助分类层，用于计算辅助损失，能起到以下作用：

  1. 防止梯度消失
  2. 充当正则约束，使中间层特征也具备分类能力

Learning Rate Schedule

fixed learning rate schedule:

decrease the learning rate by 4% every 8 epochs

Data Augmentation

  1. 图像尺寸均匀分布在8%-100%之间
  2. 长宽比在[3/4, 4/3]之间
  3. Photometric distortions, 如亮度、饱和度和对比度等

Testing Tricks

Multi-crop evaluation:

144-crops

Step1: 等比例缩放短边至256，288，320，352四种尺寸（一分四）

Step2: 每个尺度上裁剪出3个正方形，左中右或者上中下三个位置（一分三）

Step3: 左上左下右上右下中心，全局resize（一分六）

Step4: 水平镜像

总共4x3x6x2=144

Model Fusion

Experiments

  1. 模型融合：多模型比单模型精度高
  2. Multi-crop: crop越多，精度越高

GoogLeNet-v2

Paper: Batch Normalization: Accelerating Deep Network Training by Reducing Internal Covariate Shift

Link: https://arxiv.org/abs/1502.03167

Milestone: 首次提出Batch Normalization, 加速deep network的训练

Abstract

  1. 每层输入的分布都会随着训练发生变换，给训练深度神经网络带来麻烦——按原文来说：slow down the training by requiring lower learning rates and careful parameter initialization, and makes it notoriously hard to train models with saturating nonlinearities.这种现象称为internal covariate shift.(但2018年的一篇论文解释了BN的作用实际上是改变了the landscape of error surface. https://arxiv.org/abs/1805.11604 )
  2. 解决ICS的方法是normalize layer inputs
  3. Batch Normalization的优点：
    • 可以用更高的learning rates，加速模型收敛
    • 可以不用精心设计权值初始化
    • BN可以看成是regularizer，减少dropout的使用

Architecture

Batch Normalization

李宏毅老师这一块讲得挺清楚，详见2022 machine learning Week5 - Batch Normalization

BN作用：

  1. accelerate the training of deep neural nets
  2. can use higher learning rates without the risk of divergence
  3. regularize the model/ don’t need dropout for regularization
  4. make it possible to use saturating nonlinearities

Inception Architecture

v1：

v2:

对v1的改进：

  1. 激活函数前加入BN层
  2. 5x5卷积替换为2个3x3卷积
  3. 第一个inception模块增加一个inception(3c)
  4. 增多了“5x5”卷积核
  5. 分辨率变化采用stride=2的卷积而不是max pool
  6. 增加9层(10-1层)，总共31层（10表示inception数量，其实就是每个inception中因为将5x5换成了两个3x3，所以每个inception相当于多了一层，减去的1为stage1中的conv层）
